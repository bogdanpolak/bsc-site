# BSC Site

Próbna strona zbudowana w oparciu o statyczny generator stron WWW - Jekyll.

![](assets/svg2/jekyll.svg)

Więcej informacji: https://jekyllrb.com/

## Uruchomienie

```sh
bundle exec jekyll serve
```
Z opcją LiveReload:
```sh
bundle exec jekyll serve --livereload
```
